# 답변 이메일 템플릿 (Email Template)

> 근거 검색·검증을 마친 최종 답변을 **고객/직원에게 보낼 이메일 형식**으로 포맷하는 템플릿.
> [grounding-instructions.md](grounding-instructions.md)의 인용 규약(§4)을 그대로 따른다.
> 답변 주체(API 백엔드 / Claude Desktop Skill)는 아래 구조를 채워 출력한다.

## 출력 구조

```
제목: [회계기준 회신] {핵심 쟁점 한 줄}

{받는 분} 님께,

{1문단: 질문 요지 재확인 + 결론 한 줄}

■ 사실관계 / 가정
- {답변의 전제가 된 사실관계}
- {정보가 부족해 가정한 부분은 '가정:'으로 명시}

■ 검토 의견
{쟁점별로 결론과 근거. 핵심 문장 끝에 각주 번호[^n]. 단계적 판단이면 번호 목록.}

■ 결론
{실무 적용 결론. 수치 계산이 있으면 계산식·입력값 포함.}

■ 유의사항
- 아래 근거 중 '(요지)' 표시 항목은 정리본 기준이며, 최종 적용 전 원문 대조를 권고합니다.
- {확인 불가 항목이 있으면 무엇을 추가로 확인해야 하는지}
- 본 회신은 제공된 사실관계에 기초하며, 최종 판단은 담당자 검토가 필요합니다.

감사합니다.
{보내는 사람 / JAYTAX}

──────────────────────────────
근거
[^1]: {기준서/조문} 문단 {번호} (요지) — {요점}. {원문 대조 권고 / 링크}
[^2]: ...
```

## 작성 규칙

1. **제목**: `[회계기준 회신]` 또는 `[세무 회신]` 머리표 + 쟁점 한 줄. 추측성 단정 표현 금지.
2. **결론 먼저**: 첫 문단에 결론을 한 줄로 제시하고, 이후 근거로 뒷받침한다.
3. **사실관계/가정 분리**: 되묻지 않고 가정한 부분은 반드시 '가정:'으로 드러낸다(grounding §2·§3).
4. **각주 = 근거**: 본문의 모든 실질 주장에 `[^n]`을 달고, 하단 '근거'에 1:1로 매칭한다.
   - 형식: `기준서명 문단번호 (요지) — 요점. 원문 대조 권고.`
   - 원문/법령 링크가 있으면 각주에 첨부(법제처 조문 링크, 또는 `accounting-standards/k-ifrs-1115.md#§31` 내부 참조).
   - 근거를 못 찾은 항목은 본문에 "확인 불가"로 적고 각주를 만들지 않는다(지어내기 금지).
5. **수치**: 계산이 있으면 입력값 출처와 계산식을 본문 또는 별도 표로 드러낸다.
6. **톤**: 정중한 실무 회신체. 단정/과장 금지, 한정어("…로 판단됩니다", "…에 해당하는 경우")로 불확실성 보존.

## 예시 (스켈레톤 → 채움)

```
제목: [회계기준 회신] 1년 구독 선수금의 수익인식 시점

홍길동 님께,

문의하신 "12개월 구독료를 계약 시 일시 수령하는 경우의 수익인식"에 대해 회신드립니다.
결론적으로, 해당 대가는 수령 시 전액 수익이 아니라 용역 제공 기간에 걸쳐 인식하는 것이 원칙입니다[^1][^2].

■ 사실관계 / 가정
- 12개월 구독 용역, 계약 시 1년치 일시 수령.
- 가정: 구독 기간 내내 고객이 동시에 효익을 얻고 소비하는 용역(예: 콘텐츠 접속)으로 봄.

■ 검토 의견
1. 받은 선수금은 이행 전 수령액이므로 계약부채로 인식합니다[^3].
2. 구독 용역은 기간에 걸쳐 이행되는 수행의무에 해당하여, 진행률(통상 기간 경과)에 따라 수익을 인식합니다[^1][^2].

■ 결론
월할(또는 일할)로 12개월에 걸쳐 균등 인식하며, 미경과분은 계약부채 잔액으로 표시합니다.

■ 유의사항
- 근거의 '(요지)' 항목은 정리본 기준이며, 적용 전 원문 대조를 권고합니다.
- 본 회신은 제공된 사실관계에 기초합니다. 최종 판단은 담당자 검토가 필요합니다.

감사합니다.
JAYTAX

──────────────────────────────
근거
[^1]: K-IFRS 제1115호 문단 35 (요지) — 고객이 효익을 동시에 얻고 소비하면 기간에 걸쳐 이행. 원문 대조 권고. (accounting-standards/k-ifrs-1115.md#§35)
[^2]: K-IFRS 제1115호 문단 39~45 (요지) — 진행률 측정으로 기간 수익 인식. 원문 대조 권고.
[^3]: K-IFRS 제1115호 문단 106~107 (요지) — 이행 전 수령액은 계약부채. 원문 대조 권고.
```
